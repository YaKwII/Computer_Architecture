Processor Design Task 2 - Truth Table to Simplified Boolean Expression

Overview
This Python program implements a complete truth table to simplified Boolean expression converter for combinational logic design. It supports 2-4 input variables and handles both SOP (Sum of Products) and POS (Product of Sums) forms with K-map simplification and validation.

Features
- Command-line argument for number of inputs (`n=2,3,4`)
- Interactive truth table input with validation
- Canonical SOP/POS equation generation
- Minterm/maxterm listing  
- Text-based K-map display (n=2,3)
- K-map grouping and Boolean simplification
- Complete validation against original truth table

Requirements
- Python 3.x (no external libraries needed)

How to Run
```bash
python task2.py 2
